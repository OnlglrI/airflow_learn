# Внешние триггеры (External Triggers)

Обратите внимание, что **DAG Run** можно создать вручную через CLI.
Для этого выполните команду:

```bash 
airflow dags trigger --exec-date logical_date run_id
```

DAG Run, созданный **вне планировщика (scheduler)**, будет связан с **временем срабатывания триггера** и отображаться в UI рядом с запланированными DAG Run.

Логическая дата, переданная в DAG, может быть указана через аргумент `-e`. По умолчанию используется **текущая дата в часовом поясе UTC**.
