# Preset расписания (Schedule Presets)

| Preset        | Значение                                                                      | Cron          |
| ------------- | ----------------------------------------------------------------------------- | ------------- |
| `None`        | Не планировать запуск. Используется **только для DAG’ов с внешним триггером** | —             |
| `@once`       | Запустить **один раз и только один раз**                                      | —             |
| `@continuous` | Запускать **сразу после завершения предыдущего запуска**                      | —             |
| `@hourly`     | Запускать **раз в час**, в конце часа                                         | `0 * * * *`   |
| `@daily`      | Запускать **раз в день в полночь (24:00)**                                    | `0 0 * * *`   |
| `@weekly`     | Запускать **раз в неделю в воскресенье в полночь**                            | `0 0 * * 0`   |
| `@monthly`    | Запускать **раз в месяц, в полночь первого числа**                            | `0 0 1 * *`   |
| `@quarterly`  | Запускать **раз в квартал, в полночь первого дня квартала**                   | `0 0 1 */3 *` |
| `@yearly`     | Запускать **раз в год, в полночь 1 января**                                   | `0 0 1 1 *`   |

# Cron-выражения (как составлять)

Cron-выражение состоит из **5 полей**:

```
* * * * *
| | | | |
| | | | └── День недели (0–6, где 0 = воскресенье)
| | | └──── Месяц (1–12)
| | └────── День месяца (1–31)
| └──────── Час (0–23)
└────────── Минута (0–59)
```

---

## Базовые символы cron

| Символ | Значение              |
| ------ | --------------------- |
| `*`    | Любое значение        |
| `,`    | Перечисление значений |
| `-`    | Диапазон              |
| `/`    | Шаг                   |

---

## Примеры cron-выражений

### Каждые 5 минут

```text
*/5 * * * *
```

### Каждый день в 02:30

```text
30 2 * * *
```

### Каждый понедельник в 09:00

```text
0 9 * * 1
```

### По будням (Пн–Пт) в 08:00

```text
0 8 * * 1-5
```

### Каждые 3 часа

```text
0 */3 * * *
```

### 1-го числа каждого месяца в 00:00

```text
0 0 1 * *
```
